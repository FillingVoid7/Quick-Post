## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up Redis:**
   - Make sure you have Redis running locally or configure your Redis connection

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   - Visit [http://localhost:3000]
